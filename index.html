1. Learning the basics:


First and most widely used start to python is printing Hello, World!
This is one of the most basic tasks.
To do this, open your VScode and type in 'print("Hello, World")'
Next, click run in the top right of your screen.

Another thing I want to cover which is apart of the basics is variables.
Variables are like alebra, storing values.
This is like having x = 10 and having age = 20 ans so on.
These are mostly pared with inputs. Inputs are like a question which expects a value to be given.
These values can then be stored in variables. An example of this is trying to ask for your age and storing it in an age value.
You can try this by typing:'age = input("How old are you: ")'. Then after that type:'print(age)'.
This will print the value of age in the terminal.

You can also pair the variables with a print
command like if you were to ask for a name then print hello + name. You can try this by saying'age = input("What is your name: "),
then saying 'print("Hello" + name)'. Now it should ask for a name then print Hello with that name.

REMEMBER: When using the print command you should use quotation marks for words that will be specifically printed out (these are called strings).
And with words that represent a value, you shouldn't have quotation marks.


The next example is an age calculator.
It uses your birth year to calculate your age. This is made by using the time of year (2025) and subtracting your birth
year from that. The first step to doing this is to create the variable and the input text as I've shown before to get the birth year.
You do this by saying'birth_year=input("What year were you born: ")'. After you've done that you need to calculate their age.
You can then say'age = 2025 - int(birth_year). Then lastly, 'print(age)'. Now you should have your age in the terminal.

REMEMBER: Firstly, when using spaces in variables, you have to use an underscore, as the spaces can separte the words
and may even mess up your script. Secondly, when subtracting the numbers you may have noticed that I wrapped 'birth_year' ith the int function
This makes it an integer so that it is possible to subtract with another integer. Without it, it would just be a srting and the calculation would not be possible.
